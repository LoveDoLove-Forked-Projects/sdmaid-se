<i>SD Maid 2/SE</i> คือผู้ช่วยที่เชื่อถือได้ของ Android คุณ เพื่อให้เครื่องสะอาดและเป็นระเบียบ

ไม่มีใครสมบูรณ์แบบ และ Android ก็เช่นกัน

* แอปที่คุณลบไปแล้วยังทิ้งร่องรอยไว้
* ไฟล์บันทึก รายงานข้อผิดพลาด และไฟล์ที่ไม่ต้องการถูกสร้างขึ้นอยู่ตลอดเวลา
* พื้นที่จัดเก็บของคุณกำลังสะสมไฟล์และโฟลเดอร์ที่คุณไม่รู้จัก
* รูปภาพซ้ำในแกลเลอรีของคุณ

ไม่ต้องพูดถึงอีกต่อไป… ให้ SD Maid 2/SE ช่วยคุณ!

SD Maid 2/SE คือแอปและตัวจัดการไฟล์ที่รู้ว่าแอปใดสร้างไฟล์ในอุปกรณ์ของคุณ เปรียบเทียบไฟล์กับแอปที่ติดตั้งเพื่อช่วยให้คุณเพิ่มพื้นที่จัดเก็บได้อย่างปลอดภัย

✨ ทำความสะอาดหลังจากถอนการติดตั้งแอป

หากแอปสร้างไฟล์นอกโฟลเดอร์ที่กำหนด ไฟล์เหล่านั้นอาจยังคงอยู่แม้หลังจากถอนการติดตั้งแอปแล้ว เครื่องมือ "CorpseFinder" ค้นหาเศษซากของแอป บอกคุณว่าเศษซากเหล่านั้นเป็นของแอปใด และช่วยให้คุณลบออก เครื่องมือ "CorpseFinder" ค้นหาเศษซากของแอป บอกคุณว่าเศษซากเหล่านั้นเป็นของแอปใด และช่วยให้คุณลบออก

🔍 ค้นหาอุปกรณ์ของคุณอย่างชาญฉลาด

กรองโฟลเดอร์ว่าง ไฟล์ชั่วคราว แอปที่ติดตั้งแล้ว และอื่นๆ คุณสามารถสร้างเกณฑ์การค้นหาของตัวเองได้ด้วย เครื่องมือ "SystemCleaner" ค้นหาและลบไฟล์โดยอัตโนมัติตามเกณฑ์ต่างๆ

🧹 ลบไฟล์ที่ไม่จำเป็นและแคชที่ซ่อนอยู่

ภาพขนาดย่อ ถังขยะ แคชออฟไลน์ และอื่นๆ: ถ้าแอปไม่ทำความสะอาดตัวเอง แอปนี้จะทำแทน เครื่องมือ "AppCleaner" ค้นหาแอปที่มีไฟล์ไม่จำเป็น เครื่องมือ "AppCleaner" ค้นหาแอปที่มีไฟล์ไม่จำเป็น

📦 จัดการแอปทั้งหมดของคุณ

รับรายการแอปทั้งหมดที่ติดตั้งในอุปกรณ์ของคุณอย่างครบถ้วน เปิดใช้งาน ปิดใช้งาน แอปของผู้ใช้หรือแอประบบ: ไม่มีแอปใดสามารถซ่อนจากคุณได้ เครื่องมือ "AppControl" เป็นตัวจัดการแอปที่ช่วยให้คุณค้นหา จัดเรียง และควบคุมแอปของคุณ

📊 อะไรกำลังใช้พื้นที่ทั้งหมดของคุณ

การจัดการพื้นที่เก็บข้อมูลอาจซับซ้อนด้วยแอป สื่อ ระบบ และไฟล์อื่นๆ ในพื้นที่เก็บข้อมูลอุปกรณ์ การ์ด SD และอุปกรณ์ USB "StorageAnalyzer" เป็นตัวจัดการไฟล์ที่แสดงการใช้พื้นที่ในอุปกรณ์ของคุณ ทำให้การจัดการพื้นที่ง่ายขึ้น

📷 ค้นหาข้อมูลซ้ำ

ดาวน์โหลดซ้ำ รูปภาพที่ส่งผ่านโซเชียลมีเดีย หรือแค่รูปภาพที่คล้ายกันของฉากเดียวกัน: เมื่อเวลาผ่านไปสำเนาอาจสะสมมากขึ้น เครื่องมือ "Deduplicator" ค้นหาไฟล์ที่เหมือนกันหรือคล้ายกันและช่วยให้คุณลบสำเนาส่วนเกิน เครื่องมือ "Deduplicator" ค้นหาไฟล์ที่เหมือนกันหรือคล้ายกันและช่วยให้คุณลบสำเนาส่วนเกิน

👆 จัดระเบียบด้วยการปัด

เลือกโฟลเดอร์และดูทีละไฟล์ด้วยการปัดง่ายๆ เพื่อเก็บ ลบ หรือข้ามไป เลือกโฟลเดอร์และดูทีละไฟล์ด้วยการปัดง่ายๆ เพื่อเก็บ ลบ หรือข้ามไป เครื่องมือ "Swiper" ให้คุณตรวจสอบการตัดสินใจทั้งหมดก่อนที่จะลบสิ่งใด และคุณสามารถบันทึกเซสชันเพื่อดำเนินการต่อในภายหลัง

🗃️ ประหยัดพื้นที่โดยไม่ต้องลบรูปภาพ

รูปภาพอาจใช้พื้นที่จัดเก็บมาก แต่คุณไม่ได้ต้องการลบออกเสมอไป รูปภาพอาจใช้พื้นที่จัดเก็บมาก แต่คุณไม่ได้ต้องการลบออกเสมอไป เครื่องมือ "Media Squeeze" บีบอัดรูปภาพของคุณด้วยการตั้งค่าคุณภาพที่ปรับได้ เพื่อเพิ่มพื้นที่โดยยังคงรูปภาพไว้

แอปนี้ไม่มีโฆษณา แอปนี้ไม่มีโฆษณา บางฟีเจอร์ต้องอัปเกรดเป็นแบบชำระเงิน

SD Maid 2/SE เป็นผู้สืบทอดจาก SD Maid 1/Legacy
ปรับให้เหมาะกับ Android เวอร์ชันใหม่กว่า และมุ่งเน้นการทำความสะอาด

แอปนี้มีฟีเจอร์เสริมที่ใช้ AccessibilityService API เพื่อทำให้การทำงานที่น่าเบื่อเป็นอัตโนมัติ
การใช้ AccessibilityService API แอปนี้สามารถคลิกปุ่มแทนคุณเพื่อดำเนินการในหลายแอป เช่น กำลังลบแคช กำลังลบแคช
แอปนี้ไม่ใช้ AccessibilityService API เพื่อรวบรวมข้อมูล
